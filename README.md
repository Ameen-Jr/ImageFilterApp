# Image Filter App

A browser-based real-time image filtering app built with Next.js. Upload any image via drag-and-drop or file picker, apply one of nine canvas-based filters (Grayscale, Sepia, Invert, Brightness/Contrast, Fade, Cross-Process, Barrel Distortion, Ripple/Wave, Swirl), and download the result as a PNG.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Test

```bash
npm test           # Jest unit tests (19 tests)
npx playwright test  # Integration tests (8 tests)
```
